LIS System - Source Package

## Package Contents

- backend/ - Spring Boot backend source code
- frontend/ - Vue3 frontend source code
- pom.xml - Maven project configuration
- package.json - Frontend project configuration
- build.bat - Quick build script (Windows)
- build.sh - Quick build script (Linux/Mac)
- maven-dependencies/ - Core dependencies (partial)
- SOURCE_GUIDE.txt - This file (English)

## Prerequisites

- JDK 17+
- Maven 3.6+
- Node.js 18+
- MySQL 8.0

## Build Steps

### Option A: Use Build Script (Recommended)

Double-click or run:
- Windows: build.bat
- Linux/Mac: ./build.sh

### Option B: Manual Build

Backend:
cd backend
mvn clean package -DskipTests

Frontend:
cd frontend
npm install
npm run build

## Database Configuration

Edit backend/src/main/resources/application.yml:
spring:
  datasource:
    url: jdbc:mysql://YOUR_DB_HOST:3306/lisdata?useUnicode=true&characterEncoding=utf8&serverTimezone=Asia/Shanghai&useSSL=false&allowPublicKeyRetrieval=true
    username: YOUR_DB_USERNAME
    password: YOUR_DB_PASSWORD

## Quick Start

1. Create database: CREATE DATABASE lisdata;
2. Run database/schema.sql to initialize tables
3. Edit application.yml with your DB credentials
4. cd backend && mvn clean package -DskipTests
5. java -jar target/lis-open-java.jar
6. cd frontend && npm install && npm run dev
7. Access http://localhost:3000, login: admin / admin123
